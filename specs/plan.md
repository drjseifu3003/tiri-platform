# Tiri Plan

## MVP Goal
- Build a studio-first wedding invitation & event management system
- Launch before post-Easter wedding season
- Manual studio onboarding (no public signup)

## Phase 1 (MVP)
1. Studio login with phone + password
2. Invitation template management (admin global templates)
3. Event creation & linking templates
4. Guest management (add, bulk upload, check-in)
5. Media upload after event

## Phase 2 (Post-MVP)
1. Couple dashboard (view event, guest list, media)
2. RSVP tracking
3. Payment integration for studios
4. Analytics dashboard for studios

## Phase 3 (Pan-African Expansion)
1. Multi-event support (birthdays, anniversaries, corporate)
2. Multi-language, multi-culture templates
3. Partner studios across countries
